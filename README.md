### how to use
* Open devtools at [popcat.click](https://popcat.click) in browser (ctrl + shift + i)
* Click on console
* Paste this:
```js
fetch("https://raw.githubusercontent.com/qba210/popcatclick-hack/master/index.js").then((res) => res.text().then((hack) => eval(hack)))
```

now it should work!
